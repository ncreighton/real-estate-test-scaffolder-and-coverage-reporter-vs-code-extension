## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.8b5d36e8`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Real Estate Test Scaffolder and Coverage Reporter VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `8b5d36e8.enabled` | `true` | Enable/disable the extension |
| `8b5d36e8.autoRun` | `false` | Auto-run on file open |
| `8b5d36e8.logLevel` | `"info"` | Log verbosity |
